# commit message convention

| prefix                  | descrption                                                                      |
| --------------------- | ------------------------------------------------------------------------- |
| `feat: `             | 새로운 기능 추가                                                          |
| `fix: `              | 버그 수정                                                                |
| `style: `            | 코드 포맷 변경, 세미 콜론 누락, 코드 수정이 없는 경우                     |
| `refactor: `         | 프로덕션 코드 리팩토링                                                    |
| `comment: `          | 주석 추가 및 변경                                                       |
| `docs: `             | 문서 수정                                                              |
| `chore: `            | 빌드/패키지                                                            |
| `rename: `           | 파일/폴더 수정 및 이동                                                   |
| `remove: `           | 파일/폴더 삭제                                                          |
